# Suzuki Jimny Sierra JB74W 5型 Manual [Russian]

Неофициальная русскоязычная электронная версия руководства Suzuki Jimny / Jimny Sierra JB74W.

Сайт публикуется через GitHub Pages и собирается на базе MkDocs Material.

## Локальный запуск

```bash
uv sync
uv run mkdocs serve -f mkdocs.dev.yml
```

Или через Task:

```bash
task serve
```
